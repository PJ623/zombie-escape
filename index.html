Zombie Escape

written by PJ623

Levels Completed: 0

Instructions:

  • Use WASD to move.
  • The enemies move after you move.
  • You can kill enemies by moving into them, and enemies can kill you by moving into you.
  • Win the game by reaching each level's exit point.

Legend:

  • O = You
  • 0 = Enemy
  • # = Wall
